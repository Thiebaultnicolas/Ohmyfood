
mixin ? 
Une mixin est un morceau de code paramétrable et réutilisable 
n'importe où dans votre fichier SASS. 
Comme pour une fonction, une mixin peut prendre des paramètres.

comportement en cascade css ? 
la cascade des styles signifie que l'ordre d'apparition des règles 
dans le CSS a une importance ; quand deux règles applicables ont la 
même spécificité, c'est la dernière déclarée qui sera utilisée pour 
la mise en forme
